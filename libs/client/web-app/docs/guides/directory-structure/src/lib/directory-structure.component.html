Directory Structure

This project directory structure might look overwhelming and intimidating at first, but following this page and giving a bit of time to understand it before jumping right into code will help immensely.

This project was built around the idea of multi-purpose and multi-layout. You can think of this project as a Starter kit and a guide rather than just a simple template. The purpose of this project is not only to provide pre-made styles for visual elements but is also to be a guide to follow while building an app.

It's more of an answer to questions like Where should I put this file? or Which file should I put this piece of code into? rather than just a compilation of example pages and ready to use styles.
